Introduction to Web Development

Part 1

This is the first in a series of posts aimed at introducing Web Development. Throughout the series, I will discuss HTML, CSS, and Javascript starting from the basics in each category. These tutorials are perfect for an aspring web developer, a non programmer looking to get their feet wet, and even the programmer who may need a website for various projects. So let's get started!

Today we are starting with HTML. For those of you that don't know, web development is full of acronyms, so be prepared for lots of confusing letters. HTML means hyperText markup language. Online you will see lots of references to HTML5. This is merely the newest standards for HTML and we will be using it here. Your HTML is the place to put all of the content of your website including text, images, and videos. Then your CSS styles it and give it the proper look. Finally, your JavaScript adds functionality to your page. We will get to the latter two, but are starting with HTML.

Creating an HTML page is easy. First, open a text editor. This can be as simple as notepad, but I suggest downloading Sublime Text. Next, make a new file and save it with a .html extension (ex: my_first_page.html). Now we need to add some content. Here is an example of the classic 'Hello World' example. 

       
<!DOCTYPE>
<html>
 <head>
   <title>My First Website</title>
</head>

<body>
  <p>Hello World</p>
</body>
</html>

Let's talk about what each of these tag actually mean. A tag in html is each thing in between a < and >. In this case we are using the html, head, title, body, and p tags. We will get to many more html tags. You may notice that each tag comes in pairs, the second of which has a '/'. This indicates the opening and closing of tags. In other words where the tag element starts and where the content for it ends.

  • !DOCTYPE

    This tag defines that we are using HTML5. Stick it at the top of pages and you will be able to use all of the fancy tags we are going to talk about.
